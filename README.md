# SimpleCR

 kuka's simple crosshair overlay (v0.4.2.0)

### shortcuts
- CTRL + F1 - toggle show/hide
- CTRL + F2 - size: 1px, 2px x 2px
- CTRL + F3 - change colors: white, magenta, green, red, and blue
- CTRL + F4 - exit
- CTRL + PAGE UP/PAGE DOWN - adjust height
- CTRL + HOME - reset height

keyboard binds appear on tray icon hover
